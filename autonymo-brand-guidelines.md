# Autonymo — Brand Guidelines

**Version 1.0 · February 2026**

The single source of truth for Autonymo's visual identity. Use this document to create consistent, on-brand material across every touchpoint — from website and proposals to social media, diagrams, and video content.

---

## 01 — Brand Essence

*What Autonymo stands for and how we communicate it.*

### Positioning

Autonymo is a premium AI Studio that helps businesses integrate AI into their operations. We build custom operating systems — not generic tools. We're consultative, hands-on, and results-driven.

### Personality

Technically capable but not intimidating. Modern but not trendy. Confident but not arrogant. We're the senior advisor who also rolls up their sleeves.

### Voice

Clear, direct, professional. We avoid jargon unless speaking to technical audiences. We lead with outcomes, not features. We show, don't tell.

### Audience

Business owners, decision-makers, and operations managers. They need to trust us before they buy. Our brand must communicate credibility and competence instantly.

## 02 — Color System

*Our palette balances warm sophistication with strategic pops of blue. The warm tones build trust; the blue signals innovation.*

### Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **Charcoal** | `#1A1A1A` | Headlines, primary buttons, dark sections, proposal headers, video backgrounds |
| **Warm Cream** | `#F5F0EB` | Primary background, page backgrounds, light sections, diagram backgrounds |
| **Accent Blue** | `#2D7FF9` | Links, highlights, accent buttons, diagram accent nodes, data visualizations |

### Supporting Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **White** | `#FFFFFF` | Cards, surfaces |
| **Sand** | `#E8E2DA` | Dividers, subtle fills |
| **Warm Gray** | `#A39E97` | Captions, labels |
| **Text Muted** | `#6B6560` | Body text, descriptions |
| **Light Gray** | `#F9F7F5` | Alternate backgrounds |

### Functional Colors

| Color | Hex |
|-------|-----|
| **Success** | `#2D8A56` |
| **Warning** | `#D4A017` |
| **Error** | `#C0392B` |
| **Info** | `#2D7FF9` |

### Logo Gradient (reserved for logo only)

**Blue Gradient — `#0066FF` → `#00B4FF`**

Used exclusively for the Autonymo logo mark. Do not apply this gradient to other UI elements.

### Color Do's & Don'ts

**✓ Do:**

- Use Charcoal for primary text and dark sections
- Use Cream as the default page background
- Use Accent Blue sparingly for interactive elements and highlights
- Use the blue gradient only for the logo
- Pair Charcoal headers with Cream or White body sections

**✗ Don't:**

- Don't use blue as the dominant color on any page
- Don't use the gradient on buttons, backgrounds, or text
- Don't use pure white (`#FFFFFF`) as the page background — use Cream
- Don't mix warm and cool grays
- Don't introduce new colors without extending this system

## 03 — Typography

*Two fonts. One for headlines and emphasis (Plus Jakarta Sans), one for body text and UI (DM Sans). Never mix with other fonts.*

### Type Scale

| Style | Font | Weight | Size | Details |
|-------|------|--------|------|---------|
| **Display** | Plus Jakarta Sans | 700 | 48px | -4% tracking, 1.05 line-height |
| **Heading 1** | Plus Jakarta Sans | 700 | 36px | -3% tracking, 1.1 line-height |
| **Heading 2** | Plus Jakarta Sans | 600 | 24px | -2% tracking, 1.2 line-height |
| **Heading 3** | Plus Jakarta Sans | 600 | 18px | -1% tracking, 1.3 line-height |
| **Body** | DM Sans | 400 | 15px | 1.6 line-height |
| **Small** | DM Sans | 400 | 13px | 1.5 line-height |
| **Caption** | DM Sans | 500 | 11px | Uppercase, 8% letter-spacing |

### Typography Rules

**✓ Do:**

- Always use Plus Jakarta Sans for headlines and emphasis
- Always use DM Sans for body text and UI elements
- Use negative letter-spacing on headlines (-2% to -4%)
- Use uppercase + letter-spacing for labels and captions
- Keep body text at 15px minimum for readability

**✗ Avoid:**

- Never use other fonts (no Arial, Roboto, Inter, etc.)
- Never use light/thin weights for body text
- Never center-align long paragraphs
- Never use all-caps for body text or long strings
- Never exceed 70 characters per line width

## 04 — Logo

*The Autonymo "A" mark. Always uses the blue gradient. Pair it with the Autonymo wordmark in Plus Jakarta Sans Bold.*

### Logo Variations

| Variant | Background |
|---------|-----------|
| Gradient logo | On dark (`#1A1A1A`) |
| Gradient logo | On cream (`#F5F0EB`) |
| Gradient logo | On white (`#FFFFFF`) |
| White (inverted) logo | On blue (`#2D7FF9`) |

### Wordmark Lockup

Logo mark + "Autonymo" in Plus Jakarta Sans Bold.

- **Minimum clear space** around the logo equals the height of the dot in the logo mark
- **Minimum logo size:** 24px height for digital, 10mm for print

### Logo Do's & Don'ts

**✓ Do:**

- Use the gradient version on dark, cream, or white backgrounds
- Use the white version on blue or photo backgrounds
- Maintain minimum clear space around the logo

**✗ Don't:**

- Don't change the gradient colors
- Don't rotate, skew, or distort the logo
- Don't place the logo on busy or low-contrast backgrounds

## 05 — Components

*Reusable UI elements for websites, dashboards, and documents.*

### Buttons

| Type | Style | Usage |
|------|-------|-------|
| **Primary** | Charcoal background, white text | Main actions |
| **Secondary** | Transparent, outline border | Alternative actions |
| **Accent** | Blue background, white text | Use sparingly for CTAs |
| **Ghost** | Text only, no background | Tertiary / "Learn more" links |

### Tags & Badges

| Type | Style |
|------|-------|
| **Default** | Subtle gray background |
| **Blue** | Blue tint — highlight / status |
| **Dark** | Charcoal background — emphasis |

### Cards

- **Light cards:** White or Cream background, light border, `border-radius: lg (14px)`
- **Dark cards:** Charcoal background, white text, subtle border

### Spacing Scale

| Token | Value | Usage |
|-------|-------|-------|
| `xs` | 4px | — |
| `sm` | 8px | Element gaps |
| `md` | 16px | Element gaps |
| `lg` | 24px | Card padding |
| `xl` | 32px | Card padding |
| `2xl` | 48px | — |
| `3xl` | 64px | Section gaps |
| `4xl` | 96px | Section gaps |

> Use multiples of 4px for all spacing. Section gaps = 3xl–4xl. Card padding = lg–xl. Element gaps = sm–md.

### Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `sm` | 6px | Buttons |
| `md` | 10px | Inner elements |
| `lg` | 14px | Cards |
| `xl` | 20px | Sections / modals |

## 06 — Applications

*How the brand system applies across every touchpoint — website, proposals, social media, YouTube, and diagrams.*

### Website

- Cream background (`#F5F0EB`)
- Navigation: Logo left, links right
- Hero: Plus Jakarta Sans Bold headline, DM Sans body, Charcoal CTA button
- URL: autonymo.com

### Proposal / PDF

- Dark header (Charcoal) with white text in Plus Jakarta Sans Bold
- White body sections with organized rows
- Labels in uppercase, muted color
- Use blue accent for pricing highlights

### Social Media (LinkedIn)

- Rotate between dark, cream, white, and blue backgrounds
- Always include wordmark or logo
- Use Plus Jakarta Sans for headlines
- Keep layouts clean and minimal

### YouTube Thumbnails & Video

- Dark background (Charcoal)
- White text in Plus Jakarta Sans Bold
- Logo badge top-left
- Subtitle in muted text

### Diagrams & Flowcharts

- Use brand colors for diagram nodes:
  - **Charcoal** = start / primary
  - **Sand** = process
  - **Blue** = key milestone
  - **Outline** = end state
- White background, thin borders

### Email Signature

- Charcoal top-border (2px)
- Logo left-aligned
- Plus Jakarta Sans for name
- DM Sans for details
- Keep minimal
